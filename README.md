# **📂 업로드 방법**

**1. 주간 문제**

- `해당 월 + 주차 수` 폴더를 만든다.
- 해당 파일 내 파일 제목 양식은 `아이디`로 통일

```
예시) 2월 2주차 문제를 푼 경우
`Feb_2/kim429.java
```

**2. 주간 목표**

- `본인 아이디` 폴더를 만들어 코드를 업로드

```java
예시) kim429
```

- 파일 제목 양식은 `문제사이트_문제번호`로 통일

```java
예시)
BOJ_1244.java 
SWEA_3985.java
```

- 커밋메시지 `[요일]사이트_문제제목` 으로 작성

```java
예시) [월] 백준_Hello World!
```
